# YouTubey
### A YouTube inspired theme for OBS v28+ to fit the YouTube integration docks

How to install the version with the custom font:

1. Download the YouTubey.zip [here](https://github.com/derMiepz/YouTubey/releases/tag/Release)
2. Extract YouTubey.zip
3. Open YouTubey (with Font) -> FONT
4. Install the font in there
5. Move the **YouTubey** folder and **Yami_YouTubey.ovt** file into your theme folder (Default is: ``C:\Program Files\obs-studio\data\obs-studio\themes\``)
6. Restart OBS in case it's already open
7. In OBS, go to Settings -> Appearance -> Style and select the theme
8. Click OK

Enjoy your new theme!
